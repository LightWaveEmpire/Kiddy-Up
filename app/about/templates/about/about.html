A web application that assists parents in building and managing their child's routine by collecting pertinent scheduling information from multiple online sources and presenting it to the child as an age-appropriate interface.

Key features of Kiddy-Up

Automatic

Kiddy-Up provides automatic data import, intelligent data analysis and processing

Continuously

Continuously monitor for new information from the family’s online calendars and task lists.

Child Friendly

Assemble and present child-specific information in an age-appropriate format for each child.

Routine

Provide children with consistent information about their daily routines that are aligned with the family’s schedules.
